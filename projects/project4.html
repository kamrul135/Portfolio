COVID-19 Dashboard

Interactive data visualization dashboard for COVID-19 statistics using Plotly and Pandas. Real-time tracking of cases, deaths, and recoveries with interactive charts and geographical data. Deployed on Heroku for public access.

Duration: 3 weeks Domain: Data Science Type: Data Visualization

Project Overview

Developed an interactive web-based dashboard to visualize and track COVID-19 pandemic statistics in real-time. The dashboard aggregates data from multiple reliable sources and presents it through interactive charts, graphs, and geographical maps. Users can filter data by country, date range, and metrics to gain insights into the pandemic's progression. The application architecture emphasizes data accuracy, performance, and user experience.

Key Features

  • Global Coverage: Track COVID-19 statistics across 200+ countries with automatic data updates.
  • Interactive Charts: Dynamic line, bar, and pie charts built with Plotly for data exploration.
  • Geographical Maps: Heat maps and choropleth visualizations showing pandemic spread by region.
  • Advanced Filtering: Filter data by country, date range, and metric types (cases, deaths, recoveries).
  • Auto-Refresh: Automatic data synchronization with reliable COVID-19 data sources.

Technologies & Stack

Python Plotly Pandas Flask NumPy Heroku REST APIs

Technical Highlights

  • Data Processing: Efficient data cleaning and transformation using Pandas for large datasets.
  • Interactive Visualizations: Dynamic Plotly charts with hover interactions, zooming, and panning capabilities.
  • API Integration: Automated data fetching from COVID-19 APIs with error handling and retry logic.
  • Responsive Design: Mobile-friendly interface using Flask templates and Bootstrap styling.
  • Cloud Deployment: Deployed on Heroku with automated CI/CD pipeline for continuous updates.

Project Outcome

Successfully launched a comprehensive COVID-19 tracking dashboard that has served over 5,000 daily users during the pandemic. The dashboard provides accurate, real-time data visualization with an intuitive interface that makes complex epidemiological data accessible to the general public. The project demonstrates proficiency in data science, web development, and deployment of production-ready applications.
