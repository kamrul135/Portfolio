Currency Converter

Java-based currency converter application with GUI interface for live exchange rate fetching and instant conversions. Features real-time rate updates, multiple currency support, intuitive Swing-based interface, and comprehensive error handling for reliable conversions.

Duration: 2 weeks Domain: Desktop Application Type: Java GUI App

Project Overview

Developed a desktop currency converter application using Java Swing that provides real-time currency exchange rates and instant conversions between multiple currencies. The application integrates with exchange rate APIs to fetch the latest rates and displays them in an intuitive graphical interface. Users can select source and destination currencies from dropdown menus, enter amounts, and receive instant conversion results. The app includes error handling for network issues, input validation, and a clean, user-friendly design that makes currency conversion quick and easy.

Key Features

  • Multi-Currency Support: Convert between 150+ global currencies including major and minor currencies.
  • Real-Time Rates: Fetches latest exchange rates from reliable financial APIs with automatic updates.
  • Instant Conversion: Immediate calculation and display of converted amounts with high precision.
  • Intuitive GUI: Clean Swing interface with dropdown menus, text fields, and responsive buttons.
  • Swap Function: Quick swap button to instantly reverse source and destination currencies.
  • Input Validation: Robust validation to handle invalid inputs and prevent calculation errors.

Technologies & Stack

Java Swing GUI REST API JSON Parsing HttpURLConnection Exception Handling

Technical Highlights

  • Swing Components: Custom JFrame, JPanel, JComboBox, JTextField, and JButton components with event listeners.
  • API Integration: HTTP requests to exchange rate APIs using HttpURLConnection with proper error handling.
  • JSON Processing: Parsing JSON responses to extract exchange rate data efficiently.
  • Event-Driven Programming: ActionListener implementation for button clicks and user interactions.
  • Number Formatting: DecimalFormat for displaying currency values with proper precision and formatting.
  • Error Handling: Try-catch blocks for network errors, invalid inputs, and API failures.

Project Outcome

Successfully developed a fully functional currency converter application that provides accurate, real-time exchange rate conversions for over 150 currencies. The application features a clean, professional interface that makes currency conversion quick and effortless. Users benefit from instant calculations, live rate updates, and reliable performance. The project demonstrates proficiency in Java desktop application development, GUI design with Swing, API integration, and implementing user-friendly software with robust error handling.
