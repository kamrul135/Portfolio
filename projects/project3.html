CNN Image Classification

State-of-the-art Convolutional Neural Network (CNN) implemented for high-accuracy image classification on the MNIST and CIFAR-10 datasets using TensorFlow and Keras framework.

Duration: 5 weeks Domain: Deep Learning Type: Personal Project

Project Overview

Design and implementation of deep learning models to solve computer vision tasks. The project focused on building and training Convolutional Neural Networks (CNNs) to classify images into predefined categories. Starting with the MNIST dataset for handwritten digit recognition, the project scaled to the more complex CIFAR-10 dataset for object classification. The goal was to understand CNN architectures, handle overfitting, and optimize model performance.

99.2% MNIST Accuracy
88% CIFAR-10 Accuracy
10+ Layers Implemented
Fast GPU Training

Key Features

  • Custom CNN Architecture: Designed a sequential model with multiple Convolutional, Pooling, and Dense layers.
  • Data Augmentation: Implemented techniques like rotation, zoom, and horizontal flips to increase dataset diversity and prevent overfitting.
  • Visualization: Plotted training history (accuracy/loss curves) and confusion matrices to analyze model behavior.
  • Performance Optimization: Used Dropout layers and Batch Normalization to improve training stability and generalization.
  • Model Saving: Implemented callbacks for Model Checkpointing and Early Stopping.

Technologies & Stack

Python TensorFlow Keras NumPy Matplotlib OpenCV Jupyter Notebooks

Technical Highlights

  • Architecture Tuning: Experimented with kernel sizes, stride lengths, and activation functions (ReLU, Softmax).
  • Regularization: Applied L2 regularization and Dropout to mitigate overfitting on training data.
  • Optimizer Selection: Compared Adam, SGD, and RMSprop optimizers to find the fastest convergence.
  • Evaluation: Detailed classification report including precision, recall, and F1-score for each class.

Project Outcome

The developed CNN model achieved exceptional accuracy of 99.2% on the MNIST test set and a competitive 88% on CIFAR-10. The implementation of data augmentation successfully reduced the gap between training and validation accuracy. This project solidified understanding of deep learning fundamentals and practical implementation of neural networks using modern frameworks.
