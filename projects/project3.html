Project Overview

SkyMap AI is an AI-powered Weather Intelligence Platform built with React 18 that combines real-time weather data from OpenWeather API with a rule-based AI decision engine to deliver smart, actionable insights. The app features a modern sidebar navigation with 5 views (Weather, Cities, Map, Trends, Settings), an interactive AI chat assistant, 7-day forecasts, outfit recommendations, and a comfort scoring system. The architecture follows a clean service-layer pattern with 12 React components, 5 service modules, and an optional Python ML backend using FastAPI and scikit-learn.

Key Features

  • Enhanced Search: Instant city search with recent searches history, popular cities quick access, and smart input with clear functionality.
  • Real-Time Weather: Current temperature, humidity, wind speed, pressure, visibility, sunrise/sunset times, and 7-day forecast with daily highs/lows.
  • AI-Powered Insights: Comfort Score (0–100), go-outside advice, umbrella recommendations, travel-readiness checks, outfit planner, and UV protection tips.
  • Multiple Views: Weather dashboard, Cities comparison, interactive Map, Trends analysis, and Settings views with sidebar navigation.
  • AI Assistant Chat: Interactive chat interface for weather queries with context-aware responses and smart recommendations.
  • Responsive Design: Fully responsive across 4 breakpoints (1100px, 900px, 640px, 480px) with touch-optimized controls and glass-morphism UI.

Technologies & Stack

React 18 JavaScript (ES6+) CSS3 OpenWeather API React Icons Python FastAPI Scikit-learn NumPy Pandas

Technical Highlights

  • AI Decision Engine: Rule-based scoring pipeline that rates temperature, humidity, wind, visibility, and rain probability into a weighted Comfort Score (0–100).
  • Service Layer Architecture: 5 service modules (weatherApi, aiEngine, aiAssistant, mlAdapter, trendAnalyzer) cleanly separated from UI components.
  • Custom React Hooks: useWeather hook encapsulates data fetching, caching, AI insights generation, and component-service integration.
  • ML Backend (Optional): Python FastAPI server with scikit-learn model training, testing, and prediction endpoints for enhanced weather analysis.
  • Mobile-First CSS: 2,873 lines of modular CSS with CSS variables, glass-morphism effects, smooth animations, and 4 responsive breakpoints.
  • Component Architecture: 12 modular React components following Container/Presentation pattern with props-based data flow.

Project Outcome

SkyMap AI delivers a production-ready weather intelligence platform with 12 React components, 5 service modules, and an optional ML backend. The AI engine provides real-time comfort scoring, outfit recommendations, and travel-readiness assessments based on multiple weather parameters. The app is fully responsive across all device sizes with a modern glass-morphism UI, sidebar navigation, and 5 distinct views. The modular architecture allows easy extension — the AI engine can be upgraded from rule-based to ML-powered by swapping the service layer without changing any UI components.
